WC Backorder Confirmation

Woocommerce Pedido Sob Encomenda

Version 1.6.5

About This Plugin

This is a WordPress/WooCommerce plugin that enables backorder confirmation functionality for your e-commerce store. When customers attempt to order more items than available in stock, they must confirm they understand the extended delivery timeframe.

WordPress Plugin WooCommerce Extension PHP 7.4+ GPLv3 License

Features

Smart Backorder Detection

Automatically detects when order quantity exceeds available stock and shows confirmation checkbox.

Custom Email Templates

Sends specialized email notifications for backorder orders with custom messaging.

Tiny ERP Integration

Integrates with Tiny (Olist) ERP system to automatically send backorder markers.

Variable Product Support

Full support for both simple and variable products with per-variation stock tracking.

Server-Side Validation

Robust validation ensures customers cannot bypass the confirmation requirement.

Internationalization Ready

Includes Portuguese (Brazil) translations and ready for additional languages.

Requirements

  • WordPress 6.0 or higher
  • WooCommerce 4.0 or higher
  • PHP 7.4 or higher

Installation

  1. Download this plugin as a ZIP file from the repository
  2. Go to your WordPress admin panel: Plugins > Add New > Upload Plugin
  3. Upload the ZIP file and click Install Now
  4. Activate the plugin through the Plugins menu
  5. Configure Tiny ERP integration (optional) at WooCommerce > Settings > Integrations > Tiny Webhook

Configuration

Products must have stock management enabled with the option "Allow backorders and notify customer" selected. The confirmation checkbox will only appear when:

  • The product allows backorders with notification
  • The quantity selected exceeds available stock
