Plugin para permitir a realização de encomenda de produtos, onde o cliente pode realizar as encomendas após a seleção da caixa de confirmação que é exibida na própria página do produto, a caixa notificação aparece apenas em casos de o produto estar cadastrado como “Permitir encomendas e informar ao cliente” e com o estoque menor que o valor a ser encomendado, ou seja, quando a quantidade de produto selecionada é menor do que o estoque disponível no ato da compra.

É criado um template de emails específico para pedidos processando e com produtos sob encomenda, no email o cliente é informado sobre a indisponibilidade do produto em estoque e que é possível obter mais informações de disponibilidade ao entrar em contato com a loja.

É criada um aba de integração com o ERP Tiny (Olist) em page=wc-settings&tab=integration&section=tiny-webhook onde é possível informar o ID e a Descrição dos marcadores de encomenda e Amostras do ERP para o envio do marcador correspondete ao ERP, onde o marcador é enviado por meio de uma tarefa agendada. A tarefa agendada  obtém o número de id do pedido no ERP, utilizando o numeroEcommerce que é o número do pedido no Woocommerce.

É verificada a condição para aplicação do marcador onde, caso o pedido seja de amostras, então é enviado um marcador de amostras, caso contrário trata-se de uma encomenda real e nesse caso é enviado o marcador de encomeda.
